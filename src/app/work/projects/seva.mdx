# Self-Explanatory Visual Analytics for Data-Driven Insight Discovery

**Project Homepage:** [https://seva.fhstp.ac.at/en](https://seva.fhstp.ac.at/en)

Design semi-automated **visualization onboarding methods** to help users learn visual analytics tools more quickly, using **user-centered design** informed by **cognitive science**.
In the course of this project, I conducted my PhD at the TU Wien, Austria, which I finished with distinction in December 2023.
My doctoral work focused on investigating user-centered strategies for guiding analysts (biomedical scientists and data journalists) through complex visual interfaces, combining semi-automated process generation with empirical user study validation.

---

**Role:** PhD student & workpackage lead | **Timeline:** 2020 - 2023 | **Type:** FFG – IKT der Zukunft

### The Challenge
As data grows, Visual Analytics tools help users find insights through interactive visualization. However, most users lack training to understand complex interfaces, causing frustration and preventing people from using powerful analysis tools.

### My Role & Responsibilities
- Work package lead for "Visualization Design & Evaluation"
- Conceputal design of visualization & data analytics onboarding
- Coordination, conceptualization, writing, presenting scientific results at conferences
- Conducting user research, usability testing, interviews

### Approach & Strategy
**Proof-of-Concept Prototype:**
Prototyped four onboarding approaches and conducted comparative usability studies to evaluate their effectiveness. Tested abstract vs. concrete instruction strategies to understand impact on user performance and satisfaction.
The resulting prototypes can be accessed here: [https://onboarding-methods.netlify.app/](https://onboarding-methods.netlify.app)

**VisAhoi library:** Existing tools (AppCues, Intro.js, GuideChimp) block interfaces during onboarding, preventing exploration—unsuitable for visual analytics. [VisAhoi](https://github.com/datavisyn/visAhoi)solves this: a JavaScript library providing non-blocking, data-driven guidance through in-place annotations and floating action buttons. It semi-automatically generates contextual explanations of visual elements (axes, legends, mappings) using visualization data and metadata.


**Flourish Template:**
Built Flourish authoring template enabling designers to customize semi-automated onboarding through an intuitive interface—editing, reordering, removing, or adding guidance messages without coding.


### Key Complexities Addressed
- Visualization onboarding concept that is using the underlaying data of the visualization
- Application of the concept in two domains biomedical R&D and data journalism
- Semi-automated generated visualization onboarding instructions
